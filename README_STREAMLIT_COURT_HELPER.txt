Представленное приложение реализует функцию распознавания изображений в текст.
Приложение реализовано на базе web - интерфейса streamlit
Функция распознавания реализована на основе модели распознавания изображений, содержащих текстовую информацию, easyocr
Для использования в целях обеспечения работы данного приложения модель easyocr прошла дополнительное дообучение
В части используемой в данной модели сверточной нейронной сети модель дообучалась на data - setах изображений, содержащих символы кириллицы
Таким образом, использованная в данном приложении модель оптимизирована для решения задач распознавания юридических текстов судебных документов в России
Для развертывания приложения необходимо установить easyocr и streamlit через установщик pip3 на машину, с которой предполагается развертывание приложения на streamlit:
pip3 install easyocr
pip3 install streamlit
Использованная при разработке исходного кода приложения версия Python: 3.8
Приложение принимает на вход файлы изображений, содержащих текстовую информацию, формата .jpg и .png
Текстовые данные, возвращаемые приложением, могут иметь погрешности и артефакты распознавания, в силу чего нуждаются в обязательной проверке и редактировании перед использованием в проектах юридических документов
Приложение допускает образование погрешностей и артефактов распознавания по количеству прямо пропорционально степени "зашумленности" подаваемого для распознавания изображения.
Ввиду указанного выше обстоятельства, не рекомендуется использовать данное приложение для распознавания размытых, не четких изображений текстовых данных, изображений текстовых данных, полученных под углом к поверности соответствующего документа, отличным от 90 градусов, изображений низкого разрешения, сильно затемненных или осветленных изображений
